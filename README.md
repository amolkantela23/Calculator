# 📱 Calculator Plus

[![Platform](https://img.shields.io/badge/platform-Android-3DDC84.svg?style=flat-square&logo=android)](https://www.android.com)
[![Kotlin](https://img.shields.io/badge/kotlin-1.9.0-7f52ff.svg?style=flat-square&logo=kotlin)](https://kotlinlang.org)
[![Compose](https://img.shields.io/badge/Jetpack%20Compose-1.5.0-4285F4.svg?style=flat-square&logo=jetpackcompose)](https://developer.android.com/jetpack/compose)
[![License](https://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](LICENSE)

A high-performance, feature-rich Android calculator application designed with a focus on precision, utility, and modern aesthetics. Inspired by the sleek Samsung One UI, **Calculator Plus** seamlessly integrates a standard scientific calculator with a versatile unit conversion engine and a persistent history management system.

---

## ✨ Key Features

### 🔢 Advanced Calculator
- **Precision Engine**: Built on `exp4j` for accurate mathematical expression parsing.
- **Real-time Feedback**: Instant result preview as you type.
- **Smart Syntax**: Intelligent handling of parentheses, percentages, and sign toggling (+/-).
- **Dynamic UI**: Auto-scaling typography that adapts to long expressions to maintain readability.

### 🔄 Universal Unit Converter
A comprehensive conversion suite supporting 7 essential categories:
- **📏 Length**: Meters, Centimeters, Millimeters, Kilometers, Inches, Feet.
- **⚖️ Mass**: Kilograms, Grams, Milligrams, Tons, Pounds.
- **🌍 Area**: Square Meters, Square Centimeters, Square Kilometers, Hectares.
- **🧪 Volume**: Liters, Milliliters, Cubic Meters, US Gallons.
- **⏳ Time**: Hours, Minutes, Seconds, Days.
- **🚀 Speed**: Meters/Second, Kilometers/Hour, Miles/Hour.
- **💾 Data**: Megabytes, Kilobytes, Gigabytes, Terabytes.

### 📜 History Management
- **One-Tap Recall**: Instantly reload previous calculations into the active editor.
- **Contextual Access**: Accessible via a modern Modal Bottom Sheet.
- **Clean Slate**: Dedicated function to clear history logs.

---

## 🛠 Tech Stack & Architecture

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose) - A fully declarative UI approach.
- **Design System**: [Material 3](https://m3.material.io/) - Utilizing the latest Material Design components and dynamic color principles.
- **Math Engine**: [exp4j](https://www.objecthunter.net/exp4j/) - Robust expression evaluation.
- **Architecture**: Modern Android development practices using State Hoisting and Unidirectional Data Flow (UDF).

## 🚀 Getting Started

### Prerequisites
- Android Studio Ladybug (2024.2.1) or higher.
- JDK 17+.
- Android Device/Emulator running API 24+.

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/amolkantela23/Calculator-App.git
   ```
2. **Open the project**
   Launch Android Studio and select `Open`, then navigate to the cloned directory.
3. **Sync and Run**
   Allow Gradle to sync dependencies and press the **Run** button.
